Task 3

This project is a responsive replication of an award submission form web page design.  
The main goal is to create a pixel-perfect layout matching the provided reference image using HTML and CSS only.


Features

Responsive Design:
- Two-column layout on desktop (left content + right form)
- Single-column layout on smaller screens

Form UI Elements:
- Name, Email Address, Organisation input fields
- Award selection section with checkboxes
- Description & Results textareas for each award

Custom Aesthetic:
- Dark red theme background (#98242A)
- White bordered input fields and textareas
- Large “Submission” heading
- Styled submit button with ✓ icon
- Diagonal bottom-right corner design using CSS pseudo-element (::after)

---

Technologies Used

- HTML5 (Structure)
- CSS3 (Flexbox, pseudo-elements, responsive media queries)
