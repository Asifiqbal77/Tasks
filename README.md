\# Interntasks Project



\## Project Overview

This repository contains two tasks demonstrating HTML and CSS skills:



\- \*\*Task 1:\*\* Basic layout page with header, navigation, sidebars, main content, and footer.  

\- \*\*Task 2:\*\* Responsive landing page for a mobile app called Catchub showcasing app features, screenshots, and download buttons.



---



\## Task 1 - Basic Layout Page



\### Project Description

This task demonstrates a simple HTML layout structure using \*\*header, navigation, main content, sidebars, and footer\*\* with inline CSS styling.



\### Project Setup Steps

1\. Open `Task1/index.html` in any modern web browser (Chrome, Firefox, Edge).  

2\. Explore the layout including header, navigation menu, left and right content sidebars, main content area, and footer.



\### Features Implemented

\- Header section with centered title.  

\- Navigation menu below the header.  

\- Two sidebars (left and right) for additional content.  

\- Main content area between the sidebars.  

\- Footer section cleared below all content.



\### Technologies Used

\- HTML5  

\- CSS (inline styling)  



---



\## Task 2 - Catchub Landing Page



\### Project Description

This task is a \*\*responsive landing page for a mobile app\*\* called Catchub. It displays app features, screenshots, and download buttons for Play Store and App Store. The layout adapts to different screen sizes using CSS Flexbox.



\### Project Setup Steps

1\. Ensure the `Task2/images` folder is present with all required images.  

2\. Open `Task2/index.html` in a modern web browser.  

3\. Resize the browser window to see the responsive behavior.



\### Features Implemented

\- Header with logo, title, subtitle, and app download buttons.  

\- Hero image section showcasing app screenshots.  

\- Feature sections highlighting \*\*Channels\*\*, \*\*Messages\*\*, and \*\*Future updates\*\*.  

\- Responsive design for mobile and desktop screens.  

\- Footer with app download buttons and contact link.



\### Technologies Used

\- HTML5  

\- CSS3 (Flexbox, responsive styling)
